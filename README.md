# DTXManiaRS

Modern DTX rhythm-game client.

## Install desktop game

From repository root:

```sh
cargo install --path app/dtxmaniars-desktop --bin dtxmaniars --locked
```

Launch:

```sh
dtxmaniars
```

Cargo installs the binary to `~/.cargo/bin`. Ensure that directory is on your `PATH`.
