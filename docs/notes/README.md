# Notes scratchpad

Cross-agent research dumps, session logs, findings.

- `docs/notes/local/` is gitignored (per-agent scratch).
- Files in this directory **are** committed — keep them useful for other agents.

## Conventions

- One topic per file: `lane-rendering-2026-06-23.md`.
- First line: `# <Topic> — <date>`.
- Link reference files as `references/...:L<line>`.
- Mark speculations explicitly: `// spec, verify against CStagePerfCommonScreen.cs`.
