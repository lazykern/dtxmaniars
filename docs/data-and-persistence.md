# Data and Persistence

Purpose: canonical ownership map for player data, formats, backup, and safe
recovery.

Audience: players backing up or troubleshooting data and contributors changing
persistence code.

Status: Maintained. Exact verified paths and schemas are being completed in
Cycle 8.

Neighboring guides: [player guide](player-guide.md),
[compatibility](compatibility.md), [contributing](contributing.md), and
[roadmap](roadmap.md).

## Guide ownership

This guide owns configuration, native score history, compatible `score.ini`,
library preferences, input/lane profiles, layout data, version fields, write
atomicity, backup, safe deletion, and recovery. Gameplay instructions stay in
the [player guide](player-guide.md).
